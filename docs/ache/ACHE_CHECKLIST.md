# UNDERSTANDING: ACHE vs Shell & Tube

ACHE (Fin Fan) uses AIR to cool process fluid, NOT water/shell.

## ACHE STRUCTURE:
```
        TUBE BUNDLE (Finned Tubes)
               |
    +----------v----------+
    |   PLENUM CHAMBER    | <- S25139-5A IS THIS!
    |  +------+ +------+  |
    |  | FAN  | | FAN  |  |
    |  +------+ +------+  |
    +---------------------+
               |
        SUPPORT STRUCTURE
               |
    PLATFORMS, LADDERS, WALKWAYS
```
