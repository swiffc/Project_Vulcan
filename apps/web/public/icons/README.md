# PWA Icons

This folder should contain the following PNG icons for the PWA:

## Required Icons

| File | Size | Purpose |
|------|------|---------|
| icon-72.png | 72x72 | Small icon |
| icon-96.png | 96x96 | Shortcut icons |
| icon-128.png | 128x128 | Medium icon |
| icon-144.png | 144x144 | Windows tile |
| icon-152.png | 152x152 | iOS icon |
| icon-192.png | 192x192 | Android icon, Apple touch icon |
| icon-384.png | 384x384 | Large icon |
| icon-512.png | 512x512 | Splash screen |

## Design Guidelines

- Use the Vulcan "V" logo on a gradient background
- Background: Linear gradient from #6366f1 (indigo) to #a855f7 (purple)
- Logo: White "V" centered
- Icons should be square with safe padding (10-15%)
- Use maskable format (safe zone in center 80%)

## Generate Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Upload a 512x512 source image and it will generate all sizes.
